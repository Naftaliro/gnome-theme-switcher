# Third-Party Notices

This file documents all third-party projects whose names, metadata, or install URLs are referenced by the GNOME Theme Switcher application. **No third-party source code is included in this repository.** This application is an original work that stores theme names and `gsettings` key values as configuration data, and optionally downloads install scripts from the companion [zorinos-gnome-themes](https://github.com/Naftaliro/zorinos-gnome-themes) repository.
